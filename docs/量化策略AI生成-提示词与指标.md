# 量化策略 AI 生成：提示词与指标指南

> 用于让 AI（如牛牛AI）自动生成量化交易策略画布。支持每分钟执行、规则驱动。

---

## 一、执行频率说明

在提示词中明确写明：

```
策略执行频率：每分钟（1分钟K线）执行一次
数据周期：1分钟K线（1m）
```

---

## 二、提示词模板（可直接复制）

```
请生成一个【每分钟执行一次】的量化交易策略，要求如下：

【执行频率】每分钟执行一次，基于1分钟K线数据

【买入逻辑】请结合以下指标设计入场条件（可多选组合）：
- 均线：MA5/MA10/MA20 金叉、价格站上MA20
- 布林带：触及下轨、站稳上轨、带宽扩大/收缩
- KDJ：低位金叉（K<20或K上穿D）、超卖区
- RSI：低位金叉（RSI<30）、超卖反弹
- 量能：当前1分钟成交量 > 5分钟均量/5（或20分钟均量/20）
- ATR：波动率过滤，避免极端波动时入场

【卖出逻辑】请设计离场条件：
- 止盈：如 收盘价突破(入场价+2×ATR)
- 止损：如 收盘价跌破(入场价-1×ATR) 或 跌破MA20
- 技术信号：KDJ高位死叉、RSI高位死叉、跌破布林中轨

【输出格式】请以「条件1 + 条件2 + ... → 动作」的规则形式输出，便于写成代码或策略画布。
```

## 三、指标清单（供 AI 识别）

| 指标 | 常用参数 | 用途 |
|------|----------|------|
| **MA** | 5/10/20/60 | 趋势、金叉死叉、支撑阻力 |
| **BOLL** | (20,2) | 上下轨、带宽、超买超卖 |
| **KDJ** | (9,3,3) | 低位金叉、高位死叉、超买超卖 |
| **RSI** | 14 | 低位金叉、高位死叉、超买超卖 |
| **ATR** | 14 | 止损、止盈、波动过滤 |
| **成交量** | 1分钟量、5分钟均量、20分钟均量 | 放量确认、流动性过滤 |

## 四、示例策略描述（可直接作为提示词）

### 示例 1：RSI + 量能 + 持仓门槛

```
【每分钟执行】1分钟K线策略

买入：驱动标的1的RSI(14)在1分K上出现低位金叉（RSI<30且RSI上穿信号线），且1分K倒数第2根柱子的成交量 > 5分钟均量/5，且持仓市值<5万时，以市价买入100%可用资金。

卖出：驱动标的1的KDJ(9,3,3)在1分K上出现高位死叉（K>80且K下穿D），且1分K倒数第2根成交量 > 5分钟均量/5 时，平仓100%持仓。止损：收盘价跌破(入场价-1×ATR)时平仓。
```

### 示例 2：标普500 ETF + BOLL + KDJ

```
【标的】标普500ETF
【周期】5分钟K线

买入：5分K收盘价触及BOLL(20,2)下轨，且KDJ(9,3,3)呈现金叉时买入。
卖出：触及BOLL中轨或KDJ呈现死叉时卖出。
止损：入场价下方1倍ATR值。
```

### 示例 3：MA20 + 量能 + ATR 止损止盈

```
【进场】标的收盘价站上MA20，且当日成交量大于20日均量时进场。
【离场】收盘价跌破MA20时平仓。
【止损】收盘价跌破(买入价-2×ATR)。
【止盈】收盘价突破(买入价+3×ATR)。
```

## 五、四类标的策略组合（买入/卖出/止损/止盈）

### 1. 高波动标的策略（如小盘股、期权、加密货币）

**适用**：ATR% > 3%、日内波动大、1 分钟/5 分钟 K 线

| 维度 | 指标组合 | 规则 |
|------|----------|------|
| **买入信号** | KDJ + RSI + 量能 + ATR 过滤 | KDJ 低位金叉（K<20 且 K 上穿 D）且 RSI<35，且 1 分钟成交量 > 5 分钟均量/5，且 ATR% 未超过近 5 日均值 1.5 倍时进场 |
| **卖出信号** | KDJ + RSI + 量能 | KDJ 高位死叉（K>80 且 K 下穿 D）或 RSI>70，且量能放大时平仓 |
| **止损规则** | ATR | 收盘价跌破（买入价 - 1×ATR）时止损 |
| **止盈规则** | ATR + RSI | 收盘价突破（买入价 + 2×ATR）或 盈利>3% 且 RSI>70 时止盈 |

**一句话**：KDJ 低位金叉 + RSI<35 + 放量 + ATR 不过度放大时进场；KDJ 高位死叉或 RSI>70 时平仓；止损 1×ATR，止盈 2×ATR。

---

### 2. 大市值股票策略（如苹果、微软、茅台）

**适用**：流动性好、波动适中、趋势较稳定，5 分钟/15 分钟 K 线

| 维度 | 指标组合 | 规则 |
|------|----------|------|
| **买入信号** | MA + 量能 + MACD | 收盘价站上 MA20，且当日成交量大于 20 日均量，且 MACD 金叉或柱状图由负转正时进场 |
| **卖出信号** | MA + MACD | 收盘价跌破 MA20 或 MACD 死叉时平仓 |
| **止损规则** | ATR | 收盘价跌破（买入价 - 1.5×ATR）时止损 |
| **止盈规则** | ATR + 均线 | 收盘价突破（买入价 + 3×ATR）或 盈利>5% 且跌破 MA10 时止盈 |

**一句话**：站上 MA20 + 放量 + MACD 金叉时进场；跌破 MA20 或 MACD 死叉时平仓；止损 1.5×ATR，止盈 3×ATR。

---

### 3. 下降趋势股策略（抄底反弹）

**适用**：股价在 MA60 下方、趋势向下，博超卖反弹，快进快出

| 维度 | 指标组合 | 规则 |
|------|----------|------|
| **买入信号** | RSI + 布林下轨 + 量能 + 背离 | RSI<30 超卖且触及布林下轨，且 1 分钟成交量 > 5 分钟均量/5，且出现 RSI 底背离（价格新低 RSI 未新低）时进场 |
| **卖出信号** | RSI + 布林中轨 | RSI 反弹至 50 以上或价格触及布林中轨时平仓 |
| **止损规则** | ATR + 价格 | 收盘价跌破（买入价 - 1×ATR）或 亏损>2% 时止损 |
| **止盈规则** | 固定比例 + RSI | 盈利>3% 或 RSI>55 时止盈（快进快出，不贪） |

**一句话**：RSI<30 + 触及布林下轨 + 放量 + 底背离时进场；RSI>50 或触及中轨时平仓；止损 1×ATR 或 -2%，止盈 3%。

---

### 4. 上升趋势股策略（顺势加仓）

**适用**：股价在 MA60 上方、多头排列，回调加仓

| 维度 | 指标组合 | 规则 |
|------|----------|------|
| **买入信号** | MA + KDJ + 量能 | 价格回调至 MA10 或 MA20 附近企稳，且 KDJ 低位金叉，且成交量大于 5 日均量时进场 |
| **卖出信号** | MA + KDJ | 收盘价跌破 MA20 或 KDJ 高位死叉时平仓 |
| **止损规则** | MA | 收盘价跌破 MA20 或 买入价 - 1.5×ATR 时止损 |
| **止盈规则** | ATR + 趋势 | 收盘价突破（买入价 + 3×ATR）或 盈利>8% 且 MACD 柱状图缩短时止盈 |

**一句话**：回调至 MA10/MA20 企稳 + KDJ 金叉 + 放量时进场；跌破 MA20 或 KDJ 死叉时平仓；止损 MA20 或 1.5×ATR，止盈 3×ATR。

---

### 可直接复制的策略文案（牛牛AI 等）

**策略 1 - 高波动标的**：当 KDJ 低位金叉且 RSI<35，且 1 分钟成交量大于 5 分钟均量/5，且 ATR% 未超近 5 日均值 1.5 倍时进场；当 KDJ 高位死叉或 RSI>70 且量能放大时平仓；止损：收盘价跌破（买入价 - 1×ATR）；止盈：收盘价突破（买入价 + 2×ATR）或盈利>3% 且 RSI>70。

**策略 2 - 大市值股票**：当收盘价站上 MA20，且当日成交量大于 20 日均量，且 MACD 金叉时进场；当收盘价跌破 MA20 或 MACD 死叉时平仓；止损：收盘价跌破（买入价 - 1.5×ATR）；止盈：收盘价突破（买入价 + 3×ATR）。

**策略 3 - 下降趋势股（抄底）**：当 RSI<30 且触及布林下轨，且放量，且出现 RSI 底背离时进场；当 RSI>50 或价格触及布林中轨时平仓；止损：收盘价跌破（买入价 - 1×ATR）或亏损>2%；止盈：盈利>3% 或 RSI>55。

**策略 4 - 上升趋势股（顺势）**：当价格回调至 MA10 或 MA20 企稳，且 KDJ 低位金叉，且成交量大于 5 日均量时进场；当收盘价跌破 MA20 或 KDJ 高位死叉时平仓；止损：跌破 MA20 或（买入价 - 1.5×ATR）；止盈：收盘价突破（买入价 + 3×ATR）。

## 六、使用建议

1. **先写清楚标的**：如「标普500ETF」「特斯拉」「某A股」
2. **明确周期**：1分钟 / 5分钟 / 15分钟
3. **必须写止损**：避免策略只描述入场、没有风控
4. **量能条件**：建议加上成交量过滤，减少假信号
