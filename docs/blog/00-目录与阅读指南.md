# Stock Agent 项目博客：从背景到实现

本系列用「博客式」写法，把 **Stock Agent** 股票分析项目从「为什么要做」到「每一步怎么实现」讲清楚，尽量通俗易懂，方便按步骤阅读或按需查阅。

---

## 适合谁读

- 想了解这个项目在做什么、怎么跑起来的人  
- 想改代码、加功能，需要知道「数据从哪来、在哪算、怎么展示」的人  
- 想借鉴「技术面 + 消息面 + 财报 + LLM」做类似分析系统的人

## 建议阅读方式

- **快速了解**：读 01、02，再扫一眼 08、09。  
- **想改某一块**：按目录跳到对应步骤（03～10），每篇里会写「对应代码位置」。  
- **从零跟实现**：按 01 → 11 顺序读，每步都会交代「为什么这样设计、代码在哪、怎么串起来」。

---

## 代码仓库说明

文中提到的路径均为项目根目录下的相对路径，例如：

- `agents/full_analysis.py`：单只标的综合分析与 LLM 调用  
- `report/build_html.py`：报告 HTML 生成  
- `config/tickers.py`：选股池与代码规范化  

项目根目录见 README，启动方式：`python server.py`，服务默认在 http://127.0.0.1:8000 。
